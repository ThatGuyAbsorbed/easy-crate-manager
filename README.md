CrateManager is designed to make creating and managing crates simple. No complicated configuration files, no lengthy setup process, just add rewards, and you're ready to go.

Most crate plugins require editing configuration files or running multiple setup commands, Easy Crate Manager keeps everything in-game, making crate creation quick and intuitive.

### Features
- Easy in-game crate creation
- Configurable reward weights
- Automatic percentage display
- Crate preview menu
- Custom crate names
- really easy to use
- opening effect including particles and sounds

### How to use the plugin

**1.** to create a crate first place down a shulker of any colour and put a command block minecart in the top-left slot of the inventory. This acts as a safeguard so that only players with access to Creative mode can create crates.

![command block minecar in top left corner](https://cdn.modrinth.com/data/cached_images/17b0909537943dac2ce5b6bc0ace3b83d1afd52a.png)

**2.** Place the items you want players to be able to win into the shulker box excluding the first and last slots per row, they will show up in the preveiw the same way you placed them. An item automatically has a weight of 1 by default, to change this hold your item and run the command: /setweight <weight>
  
![example to show which slots can be used for the rewards](https://cdn.modrinth.com/data/cached_images/e10f9cd26eb93b9e4e776a0f33101ba366c68b01.png)

When you're finished, close the inventory. The shulker box will automatically be converted into a crate and you will be able to punch it to open the preview gui for the crate and operators will see an edit button at the bottom of the preview menu, clicking it reopens the original shulker so you can edit it

![preview gui](https://cdn.modrinth.com/data/cached_images/b516727f87f9c4f1b020579b23c2cad48fc18b75.png)

**3.** To add a custom key to a crate, hold the item you want to use for your crate and run the command /setcratekey then place the item directly below the command block minecart in the edit menu, Players will now need a matching item in order to open the crate. there will be a button for operators to give themselves a stack of that crate key in the preveiw

![custom key](https://cdn.modrinth.com/data/cached_images/b8172c92667e27ba05736abe4015ed081b4577be.png)

**4.** To give your crate a custom name, rename a piece of paper in an anvil and place it directly below the key slot. The paper's name will be used as the crate's display name and will appear above the crate after it has been interacted with.

![named paper](https://cdn.modrinth.com/data/cached_images/4c5870cba209c36c304a67f67f13fb85664f5fd8.png)

![crate with name](https://cdn.modrinth.com/data/cached_images/e11e58a1a63da31b705622f38b54e706f1c3d3d1.png)
